# Règles de création de playlist

## Composition
- Respecter le nombre de morceaux demandé (±0 si possible).
- Quand plusieurs genres sont demandés, répartir équitablement les morceaux
  entre eux (ex. 3 genres pour 20 titres ≈ 7/7/6).
- Maximum 2 morceaux du même artiste par playlist (sauf demande contraire).

## Énergie et cohérence
- « Énergie élevée » → privilégier tempo rapide, titres dansants, éviter les intros longues.
- Ordonner les titres pour une montée progressive puis un maintien de l'énergie.

## Nommage et description
- Conserver exactement le nom fourni par l'utilisateur s'il est entre guillemets.
- Générer une description courte (1 phrase) rappelant l'intention et les genres.

## Qualité
- Éviter les doublons (même titre / même artiste).
- Préférer des morceaux avec une popularité raisonnable, sauf demande de découverte.
